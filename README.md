# 双平面三路由高可用专线架构演示

基于 Three.js 的 3D 交互可视化，演示光传输网络中双平面、三路由的高可用专线架构，支持 1+1 并发和 1:1 主备两种保护策略的动态切换与故障模拟。

## 架构说明

```
平面A: 主路由 = 北线(Route 1)，备路由 = 中线(Route 2)
平面B: 主路由 = 南线(Route 3)，备路由 = 中线(Route 2)
```

- **双平面**：A 平面（青色）和 B 平面（橙色）互相独立，中线为共享备路由
- **三路由**：北线、中线、南线，每条路由含 4 根光纤（A/B 平面各一对收发）
- **波分设备**：每条路由两端各配一对波分设备（WDM），与路由位置对应
- **中继站**：每条路由中段设平面 A、B 各一个中继放大器

## 保护策略

| 策略 | 说明 |
|------|------|
| **1+1 双发选收** | 主备路由同时传输，接收端择优选收，无切换延迟 |
| **1:1 主备倒换** | 备路由平时空闲，主路由中断后自动切换，节省处理资源 |

## 功能

- **策略切换**：实时切换 1+1 / 1:1 保护模式
- **故障模拟**：独立控制各平面供电和各路由光缆通断
- **路由状态监控**：实时显示各路由工作/中断/闲置/争用状态
- **3D 交互**：鼠标拖拽旋转、滚轮缩放

## 快速开始

```bash
pnpm install
pnpm dev
```

浏览器访问 `http://localhost:5173`
